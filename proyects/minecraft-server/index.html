Creación de Instancias

Una instancia es un espacio de trabajo independiente donde puedes gestionar

  • Versión de Java para lanzar el servidor
  • Versión de Minecraft
  • Versión de forge con mods
  • Mundos asociados a la instancia
  • Gestión de Mods a utilizar
  • Configuraciones personalizadas del servidor

Gestión de Paquetes de Mods

Tiene un apartado central donde se almacenan todos los paquetes de mods. Al crear una instancia de puede elegir cuales usar y cuales no, sin importar en que paquete estén.

Respaldos de Mundos Integrado con GIT

Pueder hacer un respaldo de un mundo en cualquier momento. Los respaldos se hacen con GIT, por lo que puedes ver los cambios entre cada respaldo, volver a un respaldo anterior, etc. Nota: Es necesario tener GIT instalado en el sistema.

Facíl de actualizar

Hay un botón en la parte superior derecha que te permite buscar actualizaciones. Si hay una nueva versión disponible, te mostrará un botón para descargarla e instalarla. Todo el proceso es automático y no perderás ninguna configuración ni dato.

Instalación

Siga estos pasos para inciar Minecraft Server en su sistema:

  1. Descargue la versión adecuada para su sistema operativo
  2. Descomprima el archivo (es progrma es portable)
  3. Ejecute el archivo minecraft-server.exe
  4. ¡Comience a usar Minecraft Server!
